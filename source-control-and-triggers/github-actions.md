# Github Actions

Because Github Actions was built with Github in mind, most of the triggers come from Github.

Triggers:

* schedule
* workflow\_dispatch
* repository\_dispatch
* check\_run
* check\_suite
* create
* delete
* deployment
* deployment\_status
* fork
* gollum
* issue\_comment
* issues
* label
* milestone
* page\_build
* project
* project\_card
* project\_column
* public
* pull\_request
* pull\_request\_review
* pull\_request\_review\_comment
* pull\_request\_target
* push
* registry\_package
* release
* status
* watch
* workflow\_run
